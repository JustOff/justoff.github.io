Over the past years, these extensions have been distributed and updated from the Pale Moon Add-ons Site, but from now on this will be done through their own GitHub repositories.
In order to continue receiving updates for these extensions, you should either reinstall them from their repository releases section, or simply use the corresponding green install buttons above.
If for some reason you no longer want to update these extensions (not recommended!) and receive this notification when the browser starts, you can set the "extensions.justoff-migration" value to "false" using "about:config".
